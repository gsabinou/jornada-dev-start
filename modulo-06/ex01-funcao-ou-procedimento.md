# Exercício 1 -  Função ou procedimento?
## Para cada item abaixo, responda se é uma função (retorna um valor) ou um procedimento (só executa uma ação) e justifique em uma frase:
* Str() — É uma função, pois ele retorna o número em formato de string.
* QOut() — É um procedimento, afinal ele apenas executa a ação de exibir um texto no console, não retornando valor algum.
* Date() — É uma função, afinal ele retorna uma data.
* Len() — É uma função, ele retorna o tamanho da lista em forma de int.
* Upper() — É uma função, ele recebe uma string e a retorna em uppercase.
* ClearScreen() — É um procedimento, não possui retorno algum e apenas serve para limpar a tela do console.
